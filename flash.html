FRE-VIC RC Light and Sound System Web installer

This installer change the sounds to the RC LSS modules V3.X, please follow the instructions below:

  1. Connect your RC LSS device to a USB port
  2. Select the Sound you want to install
  3. Click "Connect" and then select the correct COM port. No device board found?
  4. The programming will take less than 3 minutes
  5. Restart the RC LSS
